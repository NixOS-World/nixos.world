Then modify the records.json file in the source code repository. You can add a document link in two ways:

1.

Example code:


{
"title": "example",
"description": "example",
"url": "https://example"
}

This will redirect your website.

2.

Example code:


{
"title": "example",
"description": "example",
"name": "example",
"domain": "example.com"
}

This will add a CNAME record in the nixos.world domain.

Please note that the location of your documentation entries should be placed in the appropriate place according to alphabetical order.

To ensure the usability of the documentation, I will review whether your website is neatly formatted and interactive. I will provide some excellent documentation website source code.

Please note that before the internationalization support is complete, I only intend to provide help for Chinese users. If you want to contribute to other languages, you can help this project to improve multilingual support.
